中間駅さがし

みんなが集まりやすい駅、すぐ見つかる。
集合場所もお店もクーポンも割り勘も、これ一つで。

主な機能

中間駅検索 複数人の最寄り駅から、路線データを活用して最適な集合駅を提案。所要時間モード・運賃モードに対応。
飲食店検索 200件+ 居酒屋・カフェ・ラーメン・バー・和食・イタリアン・中華・焼肉をジャンル別に検索。おすすめ順・予算順でソート。営業時間・予算・アクセス情報も表示。
クーポン 対象店舗のクーポンをアプリ内で保存・利用。行きたい駅を登録すると周辺のクーポンを自動表示。
おでかけプラン 昼からプラン・夜からプランを周辺のお店や施設から自動作成。LINEで友達に共有できます。
割り勘 誰が・誰の分を・いくら払ったかを入力するだけで最適な精算額を自動計算。結果をLINEで共有。
その他 乗換案内 / 駅情報（Wikipedia） / 神社・寺・公園・施設検索 / 行きたいリスト / 検索履歴 / 最寄りの繁華街

データ提供

飲食店・クーポン：Powered by ホットペッパー（リクルート）
経路探索：駅すぱあとWebサービス（ヴァル研究所）
駅情報：Wikipedia REST API
施設・公園：国土数値情報（国土交通省）
